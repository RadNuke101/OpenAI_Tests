For the input column data:

- The input column data consists of phone numbers in the format '+XX YYY-ZZZ-ZZZ', where XX represents the country code.
- The country codes in the input data range from 6 to 172.
- Each phone number is unique and follows the same format.

For the output column data:

- The output column data consists of the extracted country codes from the input phone numbers.
- The extracted country codes range from 6 to 172.
- Each extracted country code corresponds to the country code present in the input phone numbers.
