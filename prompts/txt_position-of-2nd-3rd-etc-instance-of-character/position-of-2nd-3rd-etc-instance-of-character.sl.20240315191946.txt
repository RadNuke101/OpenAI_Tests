Given a list of input strings in the format [['dimensions', 'operator', 'number'], ['phrase', 'operator', 'number']], determine the total number of characters in each string after applying the specified operations. Calculate the total number of characters in the dimensions string by multiplying the numbers in the dimensions and then adding the result. For the phrase string, calculate the total number of characters by multiplying the number with the number of characters in the phrase. Provide the total number of characters for each string as the output.
